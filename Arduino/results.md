# Arduino Sensor Test Results

## 실험 개요
- 코드: `sensor_test.ino`
- 목적: 서보 모터와 초음파 센서를 사용하여 거리 데이터를 각도별로 측정
- 각도 범위: 30° ~ 150° (5° 간격)
- 장치: Arduino Uno + HC-SR04 + SG90 서보

## 관찰/노트
- 서보가 좌우로 스윕하면서 거리값 측정 완료
- 손으로 장애물을 앞에 놓으면 거리값이 실시간 반영됨
- 일부 값이 반복되거나 노이즈 발생 (센서 특성상 자연스러움)
- 이 데이터는 Unity에서 레이더 포인트로 시각화 가능
